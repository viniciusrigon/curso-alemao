Por quê não consigo abrir documentos PDF?

Para abrir documentos no formato PDF (marcados com o ícone ), você deverá ter o software Adobe Acrobat® Reader™ instalado. Alguns documentos PDF requerem a versão mais recente do Acrobat (3 ou 4, em vez da versão 2.1). Você poderá obtê-lo gratuitamente no site da Adobe.

 

 


O que é PDF e Acrobat Reader?

O Adobe® Portable Document Format (PDF) é o verdadeiro padrão aberto para distribuição eletrônica mundial de documentos. Trata-se de um formato universal de arquivos que preserva todas as fontes, itens de formatação, cores e gráficos de qualquer documento original, independentemente do aplicativo e da plataforma utilizados para sua criação. Os arquivos Adobe PDF são compactos e podem ser compartilhados, exibidos, navegados e impressos exatamente da forma pretendida por qualquer pessoa que tenha um Adobe Acrobat® Reader™ gratuito.

O Adobe® Acrobat® Reader™ é um software gratuito que permite a exibição e a impressão de arquivos Adobe PDF (Portable Document Format).

(Fonte: Adobe)

O que é um arquivo "ZIP"?

Um arquivo com a extensão ".zip" é um arquivo compactado (zipado) para aumentar a velocidade de transferência de nosso web site para seu computador. É necessário ter um software instalado que trate arquivos ".zip", para descompactá-los.

Caso não possua, visite um dos endereços abaixo, onde vários programas freeware (gratuitos) e shareware estão disponíveis:

  • http://www.winzip.com/
  • http://www.pkware.com/downloads/
  • http://www.tucows.com/

Observação: Usuários de Macintosh podem descompactar os arquivos ".zip" com o software StuffIt Expander™ da Aladdin Systems.

  • http://www.aladdinsys.com/

 

 


O que é um arquivo "MP3"?

Um arquivo com a extensão ".mp3" é um arquivo de áudio compactado. Para ouvir músicas ou exemplos de pronúncia é necessário ter um software (chamado PLAYER) que toca os arquivos ".mp3".

Caso não possua, visite um dos endereços abaixo, onde vários MP3-Players freeware (gratuitos) e shareware estão disponíveis:

  • http://www.tucows.com/
  • http://ww8.terra.com.br/mp3box/players.html

Leia mais sobre o formato ".mp3" no Site da Terra:

  • http://ww8.terra.com.br/mp3box/faq1.html
