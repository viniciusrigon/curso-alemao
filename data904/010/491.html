damit

com isso; com isto; para que (cj); para (cj); afim de (que) (cj); por isso (cj); por este meio (cj) - en: so that

  • damit ist nicht gesagt - isto não quer dizer
  • ..., damit Sie nicht ... müssen - ..., para que você não precise ...
  • Gib her! Her damit! - Dá aqui!
  • Heraus damit! - Diga lá!
  • Hör auf damit! - Para com isso!; Chega! - Cut it out!; Don"t go on like that!
  • Schluss damit! - Para com isso!; Chega! - There"s an end of it!
  • Weg damit! - Fora! Abaixo! Morra! - Hence with it!
