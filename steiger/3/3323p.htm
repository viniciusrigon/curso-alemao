Estrutura dos Arquivos

Introdução

Todos os arquivos tem a mesma estrutura. Isto facilita aprender a usar, operar e modificar.

Tela "Entrar": Abrindo um arquivo, você entra nesta tela, que apresenta o índice do arquivo.
Tela "Processar": Nesta tela você tem o acesso para todos os campos de dados. Aqui você trabalha com os registros (dados): recolher, procurar, classificar etc.
Tela "Imprimir": Depois de classificar os dados na ordem desejada, mude para esta tela que apresenta os diferentes lay-outs para imprimí-los.
Tela "Listas automáticas": Nesta tela o FileMaker procura e classifica registros em ordem já preparado para estatísticas na tela ou na impressão.
Tela "Tabela": Esta tela mostra os registros em forma de uma lista.
Tela "Informações": Esta tela é para dados administrativos do arquivo e do usuário.





OBS: Para navegar entre as telas você pode usar:
  • O menu "Script"
  • Os atalhos de teclado
  • Os botões
