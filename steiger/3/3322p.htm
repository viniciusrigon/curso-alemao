Terminologia

Banco de dados

1: Arquivo (File): Imagine uma pasta contendo informações dispostas em forma de formulários, contendo linhas e colunas. Exemplo: Arquivo "400-Projet(os)"

2: Registro (Record): Conjunto de campos de dados com informações, identificando uma entrada única em um banco de dados. Exemplo: 1 projeto

3: Campo de Dados (Field): A menor unidade de informação existente em um arquivo de banco de dados. Exemplo: "No. do Projeto", "Título do Projeto"













Terminologia

Barra de Menus, Menu, Comando/Opção

1: Barra de Menus: Uma barra horizontal que contém os menus.
2: Menu: Uma barra vertical contendo opções (comandos) (3).
4: Opção (ou Comando): Pode ser ativado também pelo atalhos de teclado (5) ou botão (6).













Terminologia

Janela

1: Arquivo aberto não ativado
2: Arquivo aberto ativado
3: No. do lay-out
4: Símbolo "Livro"
5: Registros do arquivo
6: Registros encontrados, depois do comando "Procurar"
7: Mostra, se os registros estão "Classificados"
8: Nome do arquivo
9: Nome da tela
10: Botões
11: Nome do campo de dados
12: Campo de dados
