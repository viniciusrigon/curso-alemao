FileMaker

Caraterísticas

O software FileMaker é o único aplicativo de banco de dados relacional projetado para ajudá-lo a gerenciar e compartilhar informações com facilidade.

A partir de dados puros, é fácil e rápido criar bancos de dados funcionais em Macintosh, Windows 95/98, Windows NT ou Windows 3.1.

De fato, o FileMaker vem acompanhado de um conjunto completo de gabaritos desenvolvidos com qualidade profissional para proporcionar soluções instantâneas no uso comercial, educacional e doméstico.

Requisitos (Hardware e software)

Os arquivos option podem ser utilizados em ambiente PC e/ou Apple:
  • Hardware
    • PC
      • Sistema operacional: Win95/98 / Win 3.1
      • Computador: Mínimo Tipo 386, com unidade de disco flexível
      • Memória RAM: Mínimo 8 MB
    • Apple
      • Sistema operacional: Apple OS (Mínimo Versão 7.0)
      • Computador: Toda linha Macintosh com unidade de disco flexível
      • Memória RAM: Mínimo 8 MB

  • Software
    • FileMakerPro, no mínimo Versão 3.0
    • ou Versão "Trial" (Uso limitado)
