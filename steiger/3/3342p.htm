Segurança dos dados

Backup

O backup dos arquivos option protege você contra a perda de informações, caso ocorra falha do disco rígido ou você subscreva ou exclua dados acidentalmente. O uso do backup deve fazer parte da manutenção periódica de seu computador.
!!!: É recomendável fazer no mínimo 1x por semana um backup de todos os arquivos option.
